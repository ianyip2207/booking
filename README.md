# Ally Interiors

A static one-page landing site for a luxury interior design studio.

**Live site:** https://ianyip2207.github.io/booking/

![Ally Interiors](assets/screenshot.png)

## Features

- Responsive design with dark mode support
- Portfolio gallery with lightbox and filter
- Animated counters and scroll-reveal effects
- Image carousel with autoplay
- Contact form via FormSubmit

## Usage

Open `index.html` directly in a browser, or serve locally (required for the contact form):

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.
